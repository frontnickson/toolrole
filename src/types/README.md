# Types

Папка для TypeScript типов, интерфейсов и типовых определений. Централизованное место для всех типов приложения.

## Структура

```
types/
├── index.ts          # Основные типы (экспорт всех типов)
├── api.ts            # Типы для API
├── user.ts           # Типы пользователей
├── auth.ts           # Типы аутентификации
├── common.ts         # Общие типы
├── form.ts           # Типы форм
├── ui.ts             # Типы UI компонентов
├── business.ts       # Бизнес-типы
└── [domain].ts       # Типы для других доменов
```

## Правила создания типов

1. **Один файл = одна область** типов
2. **Экспорт через index.ts** - централизованный экспорт
3. **Именование** - PascalCase для интерфейсов, UPPER_CASE для констант
4. **Документация** - JSDoc комментарии для сложных типов
5. **Переиспользование** - типы должны быть модульными

## Пример типов

```typescript
/**
 * Пользователь системы
 */
export interface User {
  /** Уникальный идентификатор */
  id: string;
  /** Email пользователя */
  email: string;
  /** Имя пользователя */
  name: string;
  /** Аватар пользователя */
  avatar?: string;
  /** Роль пользователя */
  role: UserRole;
  /** Дата создания */
  createdAt: Date;
  /** Дата последнего обновления */
  updatedAt: Date;
}

/**
 * Роли пользователей
 */
export type UserRole = 'user' | 'admin' | 'moderator';

/**
 * Параметры для получения пользователей
 */
export interface GetUsersParams {
  page: number;
  limit: number;
  search?: string;
  role?: UserRole;
  sortBy?: keyof User;
  sortOrder?: 'asc' | 'desc';
}
```

## Типы файлов

### API Types
- **api.ts** - типы для API запросов и ответов
- **request.ts** - типы для HTTP запросов
- **response.ts** - типы для HTTP ответов

### Domain Types
- **user.ts** - типы пользователей
- **auth.ts** - типы аутентификации
- **order.ts** - типы заказов
- **product.ts** - типы продуктов

### Common Types
- **common.ts** - общие типы (статусы, ошибки)
- **utils.ts** - утилитарные типы
- **constants.ts** - типы констант

### Form Types
- **form.ts** - типы для форм
- **validation.ts** - типы валидации
- **input.ts** - типы для полей ввода

### UI Types
- **ui.ts** - типы для UI компонентов
- **theme.ts** - типы для тем оформления
- **layout.ts** - типы для макетов

### Business Types
- **business.ts** - бизнес-логика
- **workflow.ts** - рабочие процессы
- **reporting.ts** - отчетность

## Принципы

- **Централизация** - все типы в одном месте
- **Модульность** - типы разделены по областям
- **Переиспользование** - типы могут использоваться в разных частях
- **Документация** - подробное описание сложных типов
- **Версионирование** - совместимость при изменениях
